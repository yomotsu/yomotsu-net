モンハン 4 で作った装備

Posted : 2013-12-18

このエントリーはモンハン Advent Calendar 2013、18日目のエントリーです。

MH4 で作った装備について書こうと思いますが、せっかくなのでまずは Web テクノロジーぽいことを。

Web ブラウザー上にリオレウスとリオレイアを読み込んでみました。

どうにかこうにか、モデルデータを用意して、WebGL ( three.js ) を使って読み込んでいます。権利的に、動くものを Web 上に置けないので動画でお楽しみください。

さて、私は片手剣を使っています。昔は双剣でしたが、MH Tri で双剣がなくなってしまったのを期に、それ以来片手剣を使っています…。

というわけで本題。MH 4 で作った装備。

よく使う支援型

乗り -> 麻痺 -> 落とし穴 -> 落とし穴抜けたところに閃光玉 で拘束します。途中で味方のスタンが挟まるともう少し長くなります。

ちなみに、飛び込み切り直後は、スーパーアーマー状態で風圧と仰け反り無効です。飛び込み切りからそのまま罠を使うと、スーパーアーマー状態で目標の足元に罠を置くことができます。これで味方に太刀がいても安心して罠を使えます。

罠が効かない古龍などに使う装備

上記の装備の広域 2 だけ残してバランスよくした装備。(粉塵調合すればいいのでは…?)
