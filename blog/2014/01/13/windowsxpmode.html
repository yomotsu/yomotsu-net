Windows XP モードも 2014 年 4 月まで

Posted : 2014-01-13
  • windows

先日、Windows XP からのバージョン移行を検討していた方が、「XP モードがあるから Windows 7 を選んだ」とコメントをしているのを見ました。

確かに、Windows XP モードを利用すれば、Windows 7 内で Windows XP と完全互換の環境を利用することができます。しかし、待ってください。Windows XP モードは、オリジナルの Windows XP と完全に同じ動作をしますが、そのサポート ライフサイクル自体も、Windows XP とおなじ 2014 年 4 月 8 日で終了します。このことは、Windows 7 で Windows XP モードをインストールし、使用するの注釈内でオフィシャルの情報として記載されています。

一方で、

Windows XP モードでは、物理的なコンピューターの CD/DVD ドライブへのアクセス、プログラムのインストール、ファイルの保存や他のタスクが、Windows XP が動作しているコンピューターを使用している場合のように行えます。

とも述べられています。つまり、サポート ライフライクル終了後に、Windows XP モードを運用し続けると、これが抜け穴となり Windows 7 のローカル環境に何らかの被害にあってしまうかもしれません。

Windows XP モードや、その他の仮想環境上の Windows XP についても、サポート ライフサイクルが切れる、ということに注意しましょう。Windows XP モードは、Windows XP の延長サポート中に Windows 7 へ除々に移行するために用意されていたサービスであり、残念ながら今( 2014 年 1 月現在)からではあまり猶予はありません。

個人的には、Windows XP サポート ライフサイクル 終了を機に Windows 8 系に移行して、細かいアップデートに追従していけるといいのかなと考えています…。
